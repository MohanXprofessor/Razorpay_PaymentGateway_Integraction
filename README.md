# 💳 Spring Boot Razorpay Payment Gateway Integration

A simple and secure **Razorpay payment gateway integration** using **Spring Boot** and **Thymeleaf**.  
This project demonstrates order creation, payment checkout, and payment verification using Razorpay APIs.

---

## 🚀 Features

- Create Razorpay Orders
- Secure Payment Checkout
- Payment Verification
- Thymeleaf-based UI
- Environment-based configuration
- Clean MVC Architecture

---

## 🛠 Tech Stack

- Java 21
- Spring Boot
- Thymeleaf
- Razorpay Java SDK
- Maven
- HTML / CSS / JavaScript
